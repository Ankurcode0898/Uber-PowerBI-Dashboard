# Uber-PowerBI-Dashboard
End-to-end Power BI project analyzing Uber booking data using DAX, data modeling, and interactive dashboards to generate business insights.
Project Overview

This project presents an end-to-end Business Intelligence dashboard built using Power BI to analyze Uber ride booking data.

The dashboard provides insights into revenue trends, booking performance, customer behavior, vehicle performance, cancellations, and operational efficiency.

The goal of this project is to simulate a real-world business scenario where stakeholders use data-driven insights to improve revenue and reduce lost bookings.

💡 Key Business Insights

Auto category generates the highest revenue.

UPI is the most preferred payment method.

Peak booking hours observed between 6 PM – 9 PM.

High cancellation volume contributes to revenue loss.

Strong opportunity to convert first-time riders into repeat customers.

🚀 Business Recommendations

Improve driver allocation during peak demand hours.

Reduce cancellation rates through driver incentive programs.

Introduce customer loyalty programs for retention.

Encourage digital payments for operational efficiency.
